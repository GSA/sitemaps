+ (Sitemap) discover(host, fetcher:nil, max_entries:nil) + (Sitemap) discover(host, fetcher:nil, max_entries:nil) {|Entry| ... }

Discover, fetch and parse sitemaps from the given host.

Attempts to find and fetch sitemaps at a given host, by examining the `robots.txt` at that host, or if no sitemaps are found via `robots.txt`, checking a small number of common locations, including `sitemap.xml`, `sitemap_index.xml`, and the gzip versions of those same locations.

def self.discover(url, fetcher: nil, max_entries: nil, &block)
  fetcher ||= @default_fetcher
  unless url.is_a? URI
    url = "http://#{url}" unless url =~ %r{^https?://}
    url = URI.parse(url)
  end

  roots = _instance.discover_roots(url, fetcher)
  _instance.fetch_recursive(roots, fetcher, max_entries, &block)
end

def self.fetch(url, fetcher: nil, max_entries: nil, &block)
  fetcher ||= @default_fetcher
  unless url.is_a? URI
    url = "http://#{url}" unless url =~ %r{^https?://}
    url = URI.parse(url)
  end

  _instance.fetch_recursive(url, fetcher, max_entries, &block)
end

+ (Sitemap) parse(source)

Parse a sitemap from an XML string. Does not fail on invalid documents, but doesn't include invalid entries in the final set. As such, a non-XML file, or non-sitemap XML file will return an empty sitemap.

Parameters:

  • source (String) —

    an XML string to parse as a sitemap.

Returns:

  • (Sitemap) —

    the sitemap represented by the given XML string.
